📋 Content — Bulk Editor

One place for everything that shows on the staff portal. Pick a Type for each row: note = a tile that stays up (no date needed) · special = shows on its day; set Repeat (e.g. weekly) for "every Tuesday" · event = dated, shows on the calendar · info = a "Stuff to Know" card (Title = label, Details = the info; no date needed). Edit any cell, add rows, or import a CSV. Changed rows highlight — nothing saves until you click Save all changes.
